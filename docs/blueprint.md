# **App Name**: TipSplit

## Core Features:

- Bill Input: Allow users to input the total bill amount.
- Tip Percentage Selection: Enable users to select a tip percentage or input a custom amount.
- Split Calculation: Calculate the tip amount and total amount per person based on the number of people splitting the bill.
- Individual Contributions: Allow individual adjustments to contributions, and keep a running subtotal.
- Fairness Check: Suggest to users how the tip might be unfair, considering relative costs, by offering itself as a tool to analyze input.

## Style Guidelines:

- Primary color: Soft lavender (#D0B4DE) to create a calm, trustworthy and friendly feeling.
- Background color: Light pink (#F8E8EE) as a desaturated, lighter tint of the primary, conveying tenderness.
- Accent color: Slightly brighter violet (#B583B8) for focus elements, complementing the primary color.
- Body and headline font: 'Inter' sans-serif for clean readability.
- Use simple, rounded icons in shades of pink and purple to represent bill, tip, and people.
- Clean, spacious layout with clearly defined sections for bill input, tip selection, and split calculation.
- Smooth transitions and subtle animations when updating calculations.
